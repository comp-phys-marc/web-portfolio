Now for Something Completely Different.

This is a quantum processor that I fabricated from scratch in a nanofabrication facility at the Quantum Matter Institute.

These are nanoscale devices that confine single charge carriers so they can be operated as qubits.

The features of this device are measured in nanometers, comparable to the Fermi wavelength.

Everything is done in a cleanroom environment, so that no dust particles interfere with the fabrication process.

Scanning electron microscopy is used to visualize the nanoscale features of the device.
