The “emulation engine” is a full stack engineering project made freely available on GitHub. The engine includes tools for the design, simulation, visualization and testing of quantum algorithms. The engine is comprised of a React and Material UI based web user interface including programming and visualization tools, a web API for programmatically performing quantum experiments through web integrations, an expressive open source Python library for the construction and execution of quantum algorithms, an event-driven high-performance distributed simulation environment written in Mozilla’s low-level Rust programming language and Python, and a TensorFlow based simulator that allows experiments to be performed on Google’s Tensor Processing Units.
