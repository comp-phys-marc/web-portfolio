Jaeger

The Build

Jaeger is the name of the battlebot that my sister and I built for the first ever Great Canadian Bot Brawl which took place during the 2015 Maker Expo at Kitchener City Hall. We designed Jaeger in AutoCAD and built his chasis using plywood. We programmed an Arduino to understand the PWM signals from a radio remote control receiver and control 3 drill motors through a circuit we designed using 4 SPDT relays. My sister and I were able to talk to a reporter from The Star about our robot and were excited to be mentioned in this article about the competition.

robot_control.ino 


int pwm_1;
int pwm_2;
int pwm_3;

void setup() {

  pinMode(5, INPUT);
  pinMode(6, INPUT);
  pinMode(9, INPUT);
  pinMode(2, OUTPUT);
  pinMode(4, OUTPUT);
  pinMode(7, OUTPUT);

  Serial.begin(115200);

}

void loop() {

  digitalWrite(2, LOW);
  digitalWrite(4, LOW);
  digitalWrite(7, LOW);

  pwm_1 = pulseIn(5, HIGH, 25000);
  pwm_2 = pulseIn(6, HIGH, 25000);
  pwm_3 = pulseIn(9, HIGH, 25000);

  if(pwm_1>1000){
  digitalWrite(2, HIGH);
  }

  if(pwm_2>1000){
  digitalWrite(4, HIGH);
  }

  if(pwm_3>1000){
  digitalWrite(7, HIGH);
   }
}
